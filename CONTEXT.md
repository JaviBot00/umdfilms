## Decided Tech Stack

- **Pure HTML/CSS/JS** — no frameworks, no build tools, no dependencies
- **Data architecture:** Separate JSON files by section in `/data/`
- **Hosting:** Basic Hostinger (static HTML, no backend)
- **Fonts:** Cormorant Garamond (display) + DM Sans (body) via Google Fonts
- **Palette:** Black `#080808` · Red `#c0202a` · White
- **Aesthetic approach:** 70% cinematic visual impact / 30% commercial clarity
- **Page generator:** `generate-pages.js` (Node.js, no dependencies)
- **Local development server:** Live Server (VS Code) or `python -m http.server 8000`

## How the Data Architecture Works

### Stats Calculated Dynamically in `main.js`

```js
stats.projects = portfolio.length                              // from array
stats.team     = team.length                                   // from array
stats.years    = new Date().getFullYear() - config.brand.founded_year  // automatic
stats.location = config.schema.address_locality               // "Málaga"
```

### Dynamic Pages (one template → N pages)

- `team/alejandro.html` loads `team.js`, which reads the filename → finds `id: "alejandro"` in `team.json` → fills the entire page
- `portfolio/cautivo-malaga.html` loads `portfolio.js` → looks up `portfolio.json` → fills technical sheet + YouTube trailer + participating team + gallery

### How to Add New Content

1. Edit the corresponding JSON file
2. Run `node generate-pages.js` (generates the HTML)
3. Upload to Hostinger

## Design Decisions Made

| Decision | Justification |
|---|---|
| Red palette instead of gold | Client request |
| "Málaga" in `<em>` (red) in the About Us h2 | Client request |
| `stats.years` calculated from `founded_year` | Updates automatically each year |
| `stats.projects` = `portfolio.length` | Updates when adding projects to JSON |
| One HTML template → N pages via JS | Centralized maintenance |
| Nav and footer rendered by JS | Shared across all pages without duplicating HTML |
| Contact form → WhatsApp | No backend; data is passed as text in the WA URL |
| `LocalBusiness` schema on home | Local SEO for Málaga |
| `VideoObject` schema on project pages | SEO for videos on Google |
| `generate-pages.js` in pure Node.js (no dependencies) | No `npm install` required |
| Inline SVG social icons | No icon library dependency |
| Grain overlay in pure CSS | Cinematic texture without additional images |

- No `<noscript>` fallback; entire site content depends on JS
